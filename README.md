## 📊 项目简介

本项目用于记录和展示数据分析的完整流程，包含数据清洗、探索性分析、特征工程和可视化报告。

## 📁 项目结构
Python-Data-Analysis/
├── data/ # 原始数据
├── notebooks/ # Jupyter Notebook
├── scripts/ # Python脚本
├── outputs/ # 输出结果
│ ├── charts/ # 图表
│ └── reports/ # 报告
└── README.md

## 🔧 技术栈

| 工具 | 用途 |
|------|------|
| Pandas | 数据处理与清洗 |
| NumPy | 数值计算 |
| Matplotlib | 数据可视化 |
| Seaborn | 统计图表 |
| Jupyter | 交互式分析 |

## 📚 案例列表

### 案例1：电商销售数据分析
- 数据清洗（缺失值、异常值处理）
- 用户行为分析（RFM模型）
- 销售趋势可视化

### 案例2：房价预测数据预处理
- 特征工程（缺失值填充、编码）
- 相关性分析
- 数据分布探索

### 案例3：用户流失分析
- 数据探索性分析（EDA）
- 特征相关性热力图
- 流失因素洞察
